Storage Cabinet

Adds several tiers of Storage Cabinets to Minecraft. Each cabinet can hold a specific item type or item tag.

Note: Currently only the 1.17 and the 1.16 (Forge, Fabric and Architectury) versions are actively maintained/will receive support.

Features

  • Storage Cabinets
    - Comes in several tiers including: Wooden (90 slots), Iron (180 Slots), Gold (270 Slots), Diamond (360 Slots), and Emerald (450 Slots)
    - A single cabinet can hold a specific item type regardless of damage or NBT (I.E Enchanted books) or a specific item tag type (I.E Wool or logs)
  • Cabinet Manager
    - Creates a "network" of Storage Cabinets which can all be accessed via the Cabinet Manager's GUI.
    - The Cabinet Manager also supports inputting and outputting items from all attached cabinet's. I.E For item pipes or for using with a Storage bus.
  • Cabinet Key
    - Locks a cabinet to the currently stored item/tag in it. This lock will persist even after the cabinet is empty. Good for automatic sorting and such.
    - Right click while sneaking to lock/unlock a cabinet
    - Normal right-click on a Cabinet Manager to unlock/lock all attached cabinets. Sneaking right click to invert locked status.
  • Cabinet Upgrade Items
    - Upgrades a Storage Cabinet to be the same tier as the upgrade while preserving all stored items
  • Dolly
    - Used to pick up a single cabinet for transportation without having to empty it out
